# Supplemental Features Prompt

Update the Social Publishing & Moderation Platform to include the following advanced features. Use React, functional components, hooks, and your existing setup.

## Post Types & Engagement (Polls & Reposts)
Update the database schema and backend APIs to support interactive post types:
- **Polls:** 
  - Allow users to create a post with a main question and multiple text options.
  - Implement voting logic where users can select an option exactly once.
  - Display the poll question, selectable options, and live voting percentages on the frontend Post Card.
- **Reposts:**
  - Allow users to share/repost an existing post on their timeline.
  - Link the new post to the `original_post_id`.
  - Render the original embedded content gracefully inside the repost card on the frontend.

## Profile Enhancements & Follow System
Update the personal profile and user discovery routing:
- Add a persistent `follows` table to track follower/following relationships.
- **Profile Tabs:** Expand the User Profile page to include sub-tabs:
  - `Media`: Filters the user's posts to show only rich media (images/videos).
  - `Followers`: Lists users currently following the profile alongside quick links.
  - `Following`: Lists users the current profile actively follows.

## Explore Page Enhancements
Flesh out the Explore page with robust discovery and tracking tools:
- **Advanced Search Filters:** Add discrete frontend inputs and backend SQL queries for:
  - Start Date & End Date
  - Minimum Engagement threshold (Sum of Likes + Comments)
  - Language Selection dropdown (e.g. English, Spanish, French, German)
- **Trending Hashtags:** Dynamically display trending topics generated by the backend analytics aggregator on the right sidebar.
- **Suggested Users:** Fetch and display top active members (based on follower count and post volume) in the sidebar with quick 'Follow' action buttons.

## Compliance & Analytics
Ensure strict GDPR data handling and deeper insights:
- **Data Export:** Build `exportData` endpoints in the backend compliance controller. Allow users to request a complete dump of their historical data (posts, comments, likes, warnings) via the Compliance Hub. Support generating the export as downloadable `JSON`, `CSV`, and `PDF` blobs directly in the browser.
- **Creator Analytics:** Introduce a `Time-of-day heatmap` to the analytics view that visually buckets engagement times to help creators optimize posting schedules.

## Moderation Workflow: Account Suspensions
Harden the moderation tooling with strict account-level penalties:
- Allow `admin` and `senior_moderator` roles to issue Account Suspensions and Permanent Bans directly from the Moderation Dashboard queue.
- **Temporary Suspension (Read-Only Mode):** Suspend a user's account, preventing them from making any state-changing POST/PUT/DELETE requests (like posting or commenting) while still allowing them to read existing platform content.
- **Permanent Ban:** Flag user as completely banned. The Auth token middleware must perform a live database check to immediately reject all requests from banned users, effectively removing their access entirely.
